⚙ One-time Setup — Google OAuth

To write back to Google Sheets and upload to Drive, you need a free Google OAuth Client ID:

  1. Go to console.cloud.google.com
  2. Create a project → Enable Google Sheets API + Google Drive API
  3. Go to Credentials → Create OAuth 2.0 Client ID (type: Web application)
  4. Add https://claude.ai to Authorized JavaScript origins
  5. Copy the Client ID and paste it in the CLIENT_ID variable at the top of this file's <script>
  6. Click Sign in with Google in the sidebar → authorize → you're done!
✓ AudioStack API key and Voice ID are pre-filled from your credentials.
✓ Mech Touch show is pre-loaded with your Sheet and Drive folder.
